== Description ==

De snelle hosting check controleert hoe lang het duurt voordat uw website wordt geladen.


* Eenvoudige installatie
* De laadtijd is direct te zien onderaan je website
* Eenvoudig te testen voor meerdere themas en plugins.


DNR is vooral bekend om de  [Snelle Wordpress Hosting](https://www.dnr.nl/snelle-wordpress-hosting/ "Snelle Wordpress Hosting").  Voor meer informatie over de plugin zie: [Plugin Hosting Speed Check](https://www.dnr.nl/plugin-hosting-speed-check/ "Hosting Speed Check").

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Download de plugin.
2. Unzip het bestand.
3. Upload de complete hosting-speed-check-plugin map naar je WordPress /wp-content/plugins/ map.
4. De plugin is te zien onder elke pagina op je website.
